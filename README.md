# ForMyParents
mbti검사를 통한 어버이날 선물 추천 웹사이트


- Branch
```
└ master
    └ develop
        ├ feature/server/login
        └ feature/web/login
```
- Commit
``` bash
<feat>: <subject>

################
# feat : 새로운 기능 추가
# fix : 버그 수정
# docs : 문서 수정
# test : 테스트 코드 추가
# refactor : 코드 리팩토링
# style : 코드 의미에 영향을 주지 않는 변경사항
# chore : 빌드 부분 혹은 패키지 매니저 수정사항
################

<body>
```
